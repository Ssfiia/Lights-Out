# Lights-Out
Игра «Гасим свет» – это логическая игра, где игровое поле представляет собой квадратную сетку N×N, каждая клетка находится в одном из двух состояний: 0 (свет выключен) и 1 (свет включен).
Цель игры – выключить все огни, перевести все клетки в состояние 0 за минимальное количество ходов.
При нажатии изменяется состояние выбранной клетки и четырёх соседних (сверху, снизу, слева, справа). Если клетка светилась – она гаснет, если была тёмной – загорается.
# Загрузка игры 
Для загрузки игры необходимо скачать файлы File_io.cpp, Game_logic.cpp, Main.cpp, Header.h и в терминал ввести строку "g++ Main.cpp Game_logic.cpp File_io.cpp -o lights_out.exe". Либо вместе с вышеперечисленными файлами скачать Lights-Out.exe и просто запустить его.
